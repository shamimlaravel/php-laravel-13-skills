# Contributing to Laravel 13 Skills

Thank you for considering contributing to this project!

## How to Contribute

### Reporting Issues

If you find a bug or have a suggestion:

1. Check if the issue already exists
2. If not, create a new issue with:
   - Clear title
   - Detailed description
   - Code example if applicable
   - Laravel/PHP version

### Pull Requests

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/my-feature`
3. Make your changes
4. Add tests if applicable
5. Commit with clear messages
6. Push to your fork
7. Submit a Pull Request

### Adding New Attributes

If Laravel adds new attributes:

1. Update `SKILL.md` with the new attribute mapping
2. Add example in `src/Examples/` directory
3. Update this `CHANGELOG.md`
4. Update `README.md` supported components list

## Style Guide

- Use consistent markdown formatting
- Include both "Before" and "After" code examples
- Keep examples simple and clear
- Use PHP 8.3+ syntax
